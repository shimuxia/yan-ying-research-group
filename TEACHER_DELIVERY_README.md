## V0.22 更新（本轮：最终润色 + Google Scholar 同步管线）

- **新增 Google Scholar 数据同步管线**：GitHub Actions 定时任务调用 SerpAPI 获取 Scholar 作者数据，生成 `data/google-scholar-stats.json`（总被引 / h-index / i10-index）与 `data/papers.json`（论文列表），网页部署后自动读取并显示「最近同步」时间。说明文档见 `GOOGLE_SCHOLAR_SYNC.md`。
- **同步生效条件**：推送到 GitHub、配置 `SERPAPI_KEY` Secret 并部署到线上后生效；本地双击打开与微信单文件版均为静态预览，自动回退内置数据，不报错。这不是 Google 官方推送，也不是实时更新（当前为每周同步一次）。
- **论文区新增 Scholar 引用指标条**：部署版读到同步数据时显示总被引 / h-index / i10-index 与最近同步时间；无数据时该位置保持为空，不影响版面。
- **全站最终润色**：中文标点统一为全角（逗号、分号）；Hero 移除纸面噪点纹理，保持纯净浅色；招生区四个信息块去掉表格式边框，改为留白错落的轻量信息点；论文区版式与相邻区块对齐统一。
